## Java 生成 PDF 的 Demo 项目

HTML + 二维码 + 条形码 转 PDF 文件

HTML + Barcode To PDF

### 包含示例

- 使用 FreeMarker 生成 HTML
- 使用 zxing 生成二维码、条形码
- 把 HTML 转 PDF 文件
